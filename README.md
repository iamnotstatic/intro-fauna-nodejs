# Intro Fauna and Nodejs

#### This is an introduction to Fauna and Nodejs Using Express Source code

### Clone project
```bash
    # Clone Repo
    $ git clone `https://github.com/iamnotstatic/intro-fauna-nodejs`

    # Install Dependencies
    $ yarn install

    # Start the project
    $ yarn run dev        
```
